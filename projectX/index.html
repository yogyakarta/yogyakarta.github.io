Jogja +F Web Designer

Kami adalah desainer web-responsif dari Yogyakarta, Indonesia. Bagian dari Jogja +F Labs Experiments yang mengembangkan proyek berbasis web. Kami membuat web lebih intuitif dan spesial dengan efek animasi, full screen (pilihan), ramah mobile browser, sederhana, dan menggunakan standar web modern dengan HTML5, CSS3, jQuery, dan Javascript. Tidak ada Flash atau plugin lain pihak ketiga. Kami selalu membangun dari konsep, kami membuat web original dan futuristik. Dari Jogja untuk Indonesia.
